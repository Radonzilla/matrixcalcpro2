Help & Formulas

This calculator supports various matrix operations:

  • Addition: Adds corresponding elements of matrices A and B
  • Subtraction: Subtracts corresponding elements of matrix B from A
  • Multiplication: Computes the product of matrices A and B
  • Determinant: Calculates the determinant of matrix A
  • Inverse: Computes the inverse of matrix A
  • Transpose: Transposes the rows and columns of matrix A
  • Eigenvalues: Calculates eigenvalues of 2×2 matrix A
  • Power: Raises matrix A to the power n

To use the calculator, select the operation, set matrix size, enter values, and click Calculate.
